☁️ デバイス同期（Google スプレッドシート）

PC と iPhone のデータを同期します。初回のみ約10分の設定が必要です。

1Google スプレッドシートを新規作成
sheets.google.com で「空白のスプレッドシート」を作成
2Apps Script を開いてコードを貼り付け
メニュー「拡張機能」→「Apps Script」→ エディタの中身を全削除して下記を貼り付け
function doGet(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName('data') || SpreadsheetApp.getActiveSpreadsheet().insertSheet('data'); const val = sheet.getRange('A1').getValue(); const output = ContentService.createTextOutput(val || '{}'); output.setMimeType(ContentService.MimeType.JSON); return output; } function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName('data') || SpreadsheetApp.getActiveSpreadsheet().insertSheet('data'); const payload = JSON.parse(e.postData.contents); if (payload.action === 'save') { sheet.getRange('A1').setValue(JSON.stringify(payload.data)); sheet.getRange('B1').setValue(new Date().toLocaleString('ja-JP')); } const output = ContentService.createTextOutput('{"ok":true}'); output.setMimeType(ContentService.MimeType.JSON); return output; }
3デプロイする
「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」→ アクセスできるユーザー「全員」→「デプロイ」→ URL をコピー
4URL を入力して接続
📱 iPhone での使い方
Safari でこの HTML ファイルを開き、「ホーム画面に追加」するとアプリのように使えます。
データを共有するには「⬆️ クラウドへ送信」→ iPhone 側で「⬇️ クラウドから取得」の順に実行してください。
